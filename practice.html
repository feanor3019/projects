border: border-width border-style border-color;

solid—Draws a solid, continuous line (This is the default.)

none—Draws no border line

hidden—Draws a border line but does not display it

dashed—Draws a line of dashes

dotted—Draws a line of dots

double—Draws two lines around the element, taking up the full border width

groove—Adds a bevel to make the element appear pressed into the page

ridge—Adds a bevel to make the element appear raised above the page

inset—Adds a slight bevel to make the element appear slightly depressed

outset—Adds a slight bevel to make the element appear slightly raised
